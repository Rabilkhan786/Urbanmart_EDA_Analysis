#  UrbanMart EDA Analysis

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on the UrbanMart dataset to uncover patterns, trends, and actionable business insights.  

The objective is to analyze customer behavior, product performance, and sales trends to support data-driven decision-making.

---

##  Objectives

- Understand dataset structure and features
- Perform data cleaning and preprocessing
- Handle missing values and detect outliers
- Analyze sales distribution and trends
- Study customer purchasing behavior
- Evaluate category-wise product performance
- Generate business insights and recommendations

##  How to Run the Project

1. Clone the repository:
   git clone <"https://github.com/Rabilkhan786/Urbanmart_EDA_Analysis.git">

2. Navigate to project directory:
   cd UrbanMart_EDA_Analysis

3. Install dependencies:
   pip install -r requirements.txt

4. Launch Jupyter Notebook:
   jupyter notebook
